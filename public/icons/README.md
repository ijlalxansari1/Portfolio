# Technology Icons

Please add the following icon files to this directory:

## Technologies Section Icons:
- `python.svg` - Python logo
- `postgresql.svg` - PostgreSQL elephant logo
- `azure.svg` - Microsoft Azure logo
- `aws.svg` - AWS logo

## Tool Stack Icons:
- `confluence.svg` - Confluence logo
- `jira.svg` - JIRA logo
- `dbeaver.svg` - DBeaver logo
- `github.svg` - GitHub logo
- `chatgpt.svg` - ChatGPT logo
- `ansible.svg` - Ansible logo
- `visual-studio.svg` - Visual Studio logo

## Where to Get Icons:

1. **Simple Icons**: https://simpleicons.org/
   - Search for the technology name
   - Download as SVG

2. **DevIcon**: https://devicon.dev/
   - Search for technology
   - Download SVG version

3. **Official Brand Assets**:
   - Python: https://www.python.org/community/logos/
   - AWS: https://aws.amazon.com/architecture/icons/
   - Azure: https://www.microsoft.com/en-us/download/details.aspx?id=41937
   - PostgreSQL: https://www.postgresql.org/about/press/trademarks/

## Alternative: Use PNG/JPG
If you prefer PNG/JPG format, you can:
1. Place files as `.png` or `.jpg` in this directory
2. Update the icon paths in `app/components/Technologies.tsx`

## Note:
The component will automatically fallback to emoji/letter icons if image files are not found.
